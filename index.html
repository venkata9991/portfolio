projects

Inventory Management System

The project involves developing a fully functional e-commerce website using the Django framework and an SQLite3 database. The website will provide a platform for customers to browse and purchase electronic products. It will feature user authentication, product listings, shopping cart functionality, and secure payment processing. The goal is to create a seamless and intuitive shopping experience for customers while efficiently managing product inventory and order fulfillment.

A Novel Multimodal Alzheimer's Disease Detection

The project aims to utilize mobility data and deep learning models to identify the stage of Alzheimer's Disease (AD) patients. By analyzing the patterns and characteristics of the patients' mobility data, the models can accurately classify individuals into different stages of AD. This approach has the potential to enhance early detection and intervention for AD patients, improving their quality of life.

Clinical Data Reporting

This project involves developing a health monitoring website using the Django framework and MySQL database. Lab assistants can register new patients, enter clinical data for each patient, and analyze and view reports of the latest tests. The website will provide a comprehensive view of all patient records, facilitating efficient data reporting and analysis for healthcare professionals. The goal is to streamline the process of managing and accessing clinical data, enhancing patient care and decision-making.


A Web Based CV Generator Using Django Framework and wkhtmltopdf Convertor

The project involves creating a web-based CV generator using the Django framework and the wkhtmltopdf converter. Users can input their basic information, skills, and experience required for job applications. The CV generator instantly generates professionally designed resumes. The project leverages the wkhtmltopdf converter to convert the generated CV into a downloadable PDF format, ensuring compatibility and ease of sharing.

Analyze the Death age Difference of Right Handers with Left Handers

This project is about comparing the age at which right-handed people and left-handed people pass away. We will collect data on both groups and clean it up to ensure it's accurate. Then, we will analyze the data using statistical methods and charts to see if there are any noticeable differences in the ages at which they die. This will help us understand if handedness has any impact on how long people live.
